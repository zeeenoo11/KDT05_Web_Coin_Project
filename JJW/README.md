## Introduction

### Background

#### 1. 활용 목표

: 이번 주 수업의 주제는 웹 활용으로, 이번 주차에 배운 html상의 데이터 송수신과 DB 활용을 목표로 한다.

#### 2. 주제

: 지난 프로젝트에서 활용한 RNN 시계열 분석에 이어, 다양한 외부 변수를 고려한 비트코인 가격 예측 모델을 구축한다.

#### 3. 목표 및 방법

1. 비트코인 가격 예측 모델 구축
2. 외부 변수 탐색 및 반영
3. 정확도 향상을 위한 다양한 시도
4. 최종 모델 선정 및 예측
5. 결과 시각화 및 해석
6. 웹페이지 구축
7. DB 연동 필요

### Data

- 바이낸스 크롤링 : 문섭님
- 외부 변수 : 달러 가치, 금값, 나스닥
- 뉴스와 sns 등을 활요하면 좋겠지만 패스

### Method

- RNN
- Transformer
